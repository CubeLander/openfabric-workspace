# Runtime OCR / Symbol Dumps

这里保存早期对 runtime ELF / strings / symbols / dump 截图的 OCR 或文本化结果，
用于追溯 runtime 控制面调查。

这些文件不是 runtime 行为的稳定入口。稳定入口请优先看：

- `docs/runtime/README.md`
- `docs/runtime/control/README.md`
- `docs/runtime/data/README.md`
- `docs/vendor_reference/runtime_evidence/`

如果这里的 raw dump 与稳定 docs 冲突，优先回到 vendor source / runtime evidence
核实，再更新稳定 docs。
